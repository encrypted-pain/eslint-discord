# eslint-discord

> A Typescript ESLint config to use in your discord projects

## Cloning

```bash
git clone https://github.com/sasuke-7/eslint-discord my-discord-project
```

## Installing yarn

```bash
npm i --global yarn
```

## Using yarn to install the dependencies

```bash
yarn
```
